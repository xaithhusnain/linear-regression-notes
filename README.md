# Linear Regression Notes

This repository contains clear and beginner-friendly notes on Linear Regression, a fundamental algorithm in machine learning.

## Overview

Linear Regression is a supervised learning method used to predict continuous values by modeling the relationship between input variables (X) and an output variable (Y).

## Topics Covered

- Introduction to Linear Regression  
- Simple and Multiple Linear Regression  
- Mathematical formula and intuition  

## Purpose

These notes are designed for beginners and students who want a simple understanding of linear regression and a quick revision resource.
